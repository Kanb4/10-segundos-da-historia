# 10 Segundos da História

O Brasil em motion design — micro-animações por década em um scroll horizontal interativo.

## Sobre

Um projeto de portfólio que combina design gráfico, motion design e história brasileira. Cada década ganha uma paleta de cores, tipografia e animação únicas.

## Décadas

| Ano | Tema | Visual |
|-----|------|--------|
| 1920 | Semana de Arte Moderna | Geometria, cores primárias |
| 1950 | Copa do Mundo | Bola, torcida, ondas |
| 1960 | Bossa Nova | Ondas orgânicas, notas musicais |
| 1970 | AI-5 | Barras, opressão, resistência |
| 1985 | Diretas Já | Placas, punho, estrelas |
| 2002 | Lula Presidente | Bandeira, povos |
| 2016 | Rio Olimpíadas | Anéis, movimento |
| Hoje | O Futuro é Agora | Glows, partículas |

## Stack

- React + Vite
- GSAP + ScrollTrigger (scroll horizontal)
- CSS animations (micro-animações por década)
- Google Fonts (Playfair Display, Inter, Bebas Neue, Space Grotesk)

## Deploy

### GitHub

```bash
git remote add origin https://github.com/SEU-USER/10-segundos-da-historia.git
git branch -M main
git push -u origin main
```

### Vercel

1. Acesse [vercel.com](https://vercel.com)
2. Importe o repositório do GitHub
3. Framework: **Vite**
4. Build Command: `npm run build`
5. Output Directory: `dist`
6. Deploy!

## Desenvolvimento

```bash
npm install
npm run dev
```
